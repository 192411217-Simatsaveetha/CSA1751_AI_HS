# Breadth First Search (BFS)

## Description
This project implements the BFS algorithm using an adjacency list.

## How to Run
1. Clone the repo
2. Run `python bfs.py`
3. Input your graph and start node

## Example
Graph: {'A': ['B', 'C'], 'B': ['D'], 'C': [], 'D': []}
Start Node: 'A'
Output: ['A', 'B', 'C', 'D']
